# 🚀 راهنمای کامل راه‌اندازی AI با Cloudflare Workers

این راهنما رو **قدم به قدم** دنبال کن. کل پروسه ۱۵ دقیقه طول می‌کشه و آخرش یه چت‌بات هوشمند با امنیت کامل داری.

---

## 🎯 چی می‌خوایم بسازیم؟

```
کاربر تو سایتت → Cloudflare Worker (با کلید API مخفی) → Google Gemini → جواب
```

**نتیجه:** API Key هرگز تو مرورگر کاربرها نمیاد. کاملاً امن!

---

# 📍 بخش ۱: گرفتن API Key از Google (3 دقیقه)

### قدم ۱.۱
برو به این لینک:
👉 **https://aistudio.google.com/apikey**

با اکانت گوگلت لاگین کن.

### قدم ۱.۲
روی دکمه آبی **"Create API Key"** کلیک کن.

### قدم ۱.۳
یه کلید بهت می‌ده که شبیه این می‌مونه:
```
AIzaSyBxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

🚨 **خیلی مهم:** این کلید رو کپی کن و یه جای امن نگه دار. بعداً لازمت می‌شه. **به هیچ‌کس نشون نده!**

✅ **تموم شد بخش ۱!**

---

# 📍 بخش ۲: ساخت Cloudflare Worker (5 دقیقه)

### قدم ۲.۱: ثبت‌نام تو Cloudflare
برو به این لینک:
👉 **https://dash.cloudflare.com/sign-up**

ایمیل و پسورد بزن، اکانت بساز. (رایگانه، کارت اعتباری نمی‌خواد)

### قدم ۲.۲: رفتن به Workers
بعد از لاگین، تو منوی چپ روی **"Workers & Pages"** کلیک کن.

### قدم ۲.۳: ساخت Worker جدید
دکمه **"Create"** یا **"Create Worker"** رو بزن.

### قدم ۲.۴: انتخاب اسم
اسم Worker رو بذار:
```
moein-ai-proxy
```

دکمه **"Deploy"** رو بزن. (یه نسخه‌ی پیش‌فرض Worker با "Hello World" deploy می‌شه. اشکالی نداره، الان عوضش می‌کنیم.)

### قدم ۲.۵: ویرایش کد Worker
بعد از Deploy، روی **"Edit code"** یا **"Continue to project"** بعد **"Edit code"** کلیک کن.

یه Editor باز می‌شه با کد Hello World.

**همه‌ی کد فعلی رو پاک کن** و کد فایل `cloudflare-worker.js` رو که برات ساختم رو **پیست کن**.

### قدم ۲.۶: Save and Deploy
بالای راست روی دکمه **"Save and Deploy"** کلیک کن. کنفرم کن.

### قدم ۲.۷: کپی کردن آدرس Worker
حالا یه آدرس بهت می‌ده شبیه این:
```
https://moein-ai-proxy.YOURNAME.workers.dev
```

این آدرس رو کپی کن! بعداً لازمت می‌شه.

✅ **تموم شد بخش ۲!**

---

# 📍 بخش ۳: گذاشتن API Key مخفی تو Worker (2 دقیقه)

این قسمت **مهم‌ترین** بخشه. اینجاست که API Key رو امن می‌کنیم.

### قدم ۳.۱
تو صفحه Worker، برو به تب **"Settings"** (یا "Configuration").

### قدم ۳.۲
گزینه **"Variables and Secrets"** یا **"Variables"** رو پیدا کن.

### قدم ۳.۳
روی **"Add variable"** یا **"Add"** کلیک کن.

### قدم ۳.۴
یه فرم برات باز می‌شه. این مقادیر رو وارد کن:

| فیلد | مقدار |
|------|-------|
| **Variable name** | `GEMINI_API_KEY` |
| **Type** | **Secret** (خیلی مهم! نه "Plain text") |
| **Value** | API Key که از Google گرفتی (شروع با `AIza`) |

### قدم ۳.۵
**"Save and Deploy"** بزن.

🎉 **حالا API Key کاملاً امنه!** هیچ‌کس نمی‌تونه ببینش، حتی کسی که ادمین نیست تو Cloudflare.

✅ **تموم شد بخش ۳!**

---

# 📍 بخش ۴: آپدیت کد سایت (5 دقیقه)

### قدم ۴.۱: ویرایش `chatbot.js`
فایل `chatbot.js` رو باز کن. همون اول این بخش رو می‌بینی:

```javascript
const CONFIG = {
  WORKER_URL: ""
};
```

تو فاصله‌ی `""` آدرس Worker که تو بخش ۲ کپی کردی رو پیست کن:

```javascript
const CONFIG = {
  WORKER_URL: "https://moein-ai-proxy.YOURNAME.workers.dev"
};
```

⚠️ **توجه:** آدرس کامل رو پیست کن. بدون `/` آخر.

### قدم ۴.۲: آپلود به GitHub
این فایل آپدیت شده‌ی `chatbot.js` رو به ریپوی GitHub خودت آپلود کن (جایگزین فایل قبلی).

### قدم ۴.۳: تست!
برو به سایتت:
```
https://moeintash.github.io/Home-page/
```

از چت‌بات یه سوال بپرس مثل: *"What is your research about?"*

اگه جواب طبیعی و هوشمند گرفتی، **🎉 موفق شدی!**

اگه جواب rule-based گرفتی (مثل قبل)، یه نگاه به تنظیمات بنداز:
1. آدرس Worker تو `chatbot.js` درسته؟
2. API Key تو Worker `Secret` ذخیره شده (نه Plain Text)?
3. چند دقیقه صبر کن (DNS گاهی زمان می‌بره)

✅ **تموم شد بخش ۴!**

# 🐛 رفع مشکل

### چت‌بات همچنان rule-based جواب می‌ده
1. Console مرورگر (F12) رو باز کن، تب Console رو ببین
2. اگه نوشته `"Worker returned error: 500"`: API Key اشتباه ذخیره شده. بخش ۳ رو دوباره چک کن
3. اگه نوشته `"Worker returned error: 403"`: دامنه‌ت تو Worker مجاز نیست
4. اگه `WORKER_URL` تو `chatbot.js` خالیه: بخش ۴.۱ رو چک کن

### API Key لو رفت یا می‌خوام عوضش کنم
1. برو به https://aistudio.google.com/apikey
2. کلید قدیمی رو حذف کن
3. کلید جدید بساز
4. تو Cloudflare Worker → Settings → Variables → مقدار `GEMINI_API_KEY` رو آپدیت کن
5. هیچ تغییری تو سایت لازم نیست! ✨

### می‌خوام آمار استفاده ببینم
تو Cloudflare Dashboard → Workers → moein-ai-proxy → Metrics

# 🎁 امکانات اضافه (اختیاری)

اگه خواستی پیشرفته‌ترش کنی:

### Rate Limiting
اگه بخوای جلوی سواستفاده رو بگیری (مثلاً یه نفر ۱۰۰۰ بار سوال بپرسه):
می‌تونی تو Worker از Cloudflare KV استفاده کنی. بهم بگو، برات می‌سازم.

### Custom Domain
اگه دامنه شخصی داری، می‌تونی Worker رو به جای `workers.dev` با دامنه خودت بذاری (مثل `api.moeintash.com`).

### Logging
می‌تونی هر سوالی که از چت‌بات پرسیدن رو لاگ کنی تا ببینی چی پرسیدن (و KB رو بهتر کنی).
